# Contributing

Thanks for helping improve README Motion Gallery.

1. Create a focused branch from `main`.
2. Install development dependencies with `python -m pip install -e ".[dev]"`.
3. Run `ruff check .` and `pytest`.
4. Include tests for behavior changes.
5. Open a pull request that explains the user-visible result and any GIF size impact.

Keep version 0.1 deterministic and local-first. Remote URL fetching, hosted rendering, and automatic repository
commits require a separate design and security review.
