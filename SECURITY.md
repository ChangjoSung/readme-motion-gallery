# Security Policy

## Supported versions

Only the latest release receives security updates during the alpha period.

## Reporting a vulnerability

Please use GitHub private vulnerability reporting when it is enabled for the repository. Do not open a public
issue for vulnerabilities involving file access, decompression bombs, path handling, or GitHub Actions.

Version 0.1 intentionally accepts only local PNG and JPEG files. It does not fetch arbitrary URLs or commit
generated files automatically.
